# Year Progress

Live Demo: <https://pantsman-jp.github.io/year-prog/>

## Overview
A simple web application that visualizes the progress of the current year
based on the user's local system time.
It displays a progress bar and the remaining time
until the end of the year in days, hours, minutes, and seconds.

## Usage
1. Clone this repository:
  ```bash
  git clone https://github.com/pantsman-jp/year-prog.git
  ```
2. Open `index.html` in a web browser.
